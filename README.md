# docker-python-cronjob
Run python script as a cron job using Docker

Tested 2023-03-29

## Build Docker
```
docker build -t "python-cron:latest" .
```
## Run docker container
```
docker run -it --rm python-cron
```
## Container console logs
```
hello world!
Welcome to python cron job
hello world!
Welcome to python cron job
hello world!
```
